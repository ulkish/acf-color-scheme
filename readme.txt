Adds a color scheme field type to Advanced Custon Fields Pro.
Create your own color schemes using hex and make them available from your
admin panel. You will need to know some PHP to be able to actually change 
something using this, since this is just the front-end of the functionality, 
but if you're looking for a controller this is for you.

== Description ==

Adds a color scheme field type to Advanced Custon Fields Pro.

= Compatibility =

This ACF field type requires:

* ACF 5

== Installation ==

1. Copy the `acf-color_scheme` folder into your `wp-content/plugins` folder
2. Activate the Color Scheme plugin via the plugins admin page
3. Create a new field via ACF and select the Color Scheme type
4. Read the description above for usage instructions
